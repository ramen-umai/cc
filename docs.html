ブロック一覧

1. 論理（Logic）

  • if / if else（もし〜なら）

    条件が正しいときだけ、指定した処理を実行するブロックです。たとえば、「もし温度が30度以上なら扇風機を止める」といった動作を設定できます。

  • true / false（真・偽）

    論理値を表します。プログラムの判断基準として、条件が成り立つ（true）か成り立たない（false）かを示します。これにより、分岐処理などを明確にします。

  • =, ≠, <, >, ≤, ≥（比較）

    2つの数値や文字、変数の値を比較するためのブロックです。数が同じか、どちらが大きいか、小さいかを判断し、次の処理の条件決定に使われます。例えば、テストの点数が一定以上かどうかをチェックする場合などに利用します。

  • and / or / not（論理演算）

    複数の条件を組み合わせたり、条件を反転させたりするための演算です。
    - and：すべての条件が正しいときに真となる。
    - or：いずれかの条件が正しければ真となる。
    - not：条件の論理値を反転させる。

2. くり返し（Loops）

  • repeat n times（n回くり返す）

    指定した回数だけ同じ処理を実行します。例えば、「10回『こんにちは』と表示する」場合など、回数が決まっている処理に便利です。

  • while / until（〜の間／〜まで）

    指定した条件が成立している間（または成立するまで）処理を繰り返すブロックです。
    例として、ユーザーがボタンを押すまで待機し、ボタンが押されたら処理を中断する場合などに使用します。
    注意点として、条件が永遠に成立すると無限ループになってしまう可能性があるため、必ず条件を正しく設定する必要があります。

  • for（数つきのくり返し）

    初期値、条件、増減幅を設定して、一定の範囲内で変数の値を変えながら処理を繰り返します。
    例として、1から10までの数字を順に表示する場合や、配列の全要素にアクセスする際に用いられます。

3. 計算（Math）

  • ＋ － × ÷

    基本的な算術演算ができるブロックです。
    数値の加算、減算、乗算、除算を使用して計算を行い、結果を他のブロックに渡すことができます。
    たとえば、ゲームのスコア計算や、合計金額の計算などで利用されます。

  • ランダムな数

    指定した範囲内でランダムな数字を生成します。
    これにより、予測不可能な動作や、ゲームのランダムなイベントを作り出すことができます。
    例として、サイコロの目のシミュレーションや、抽選プログラムに利用できます。

4. 文字（Text）

  • 文字をつなげる

    複数の文字列を連結して一つの文章にします。
    例えば、「おはよう」と「ございます」をつなげることで「おはようございます」というメッセージを作れます。
    このブロックを使うことで、動的なメッセージや表示内容を作成できます。

  • 文字の長さ・一部を取り出す

    入力された文字列の長さを数えたり、特定の位置から一文字または一部を抽出します。
    これにより、ユーザーの名前の一部を抜き出したり、文字列の内容をチェックすることができます。

5. 変数（Variables）

  • 変数を作る・使用する・値を変更する

    データを一時的に保存するための箱のようなものです。
    プログラム内で必要な数値、文字列、論理値などを格納し、何度も利用することができます。
    たとえば、ゲームの得点やユーザーの入力情報を保存・更新する場合に使われます。

6. 関数（Functions）

  • 関数を作る

    よく使う処理をひとまとまりにして、一つの名前をつけるブロックです。
    これにより、同じ処理を何度も書かずに済み、プログラムが分かりやすくなります。
    例えば、複雑な計算処理や一連の動作を関数にまとめ、必要に応じて呼び出すことが可能です。

  • 関数を呼び出す

    作成した関数を実際に実行するブロックです。
    関数に設定された処理内容が実行され、計算や動作が行われます。
    これにより、処理を再利用してコードの重複を減らすことができます。
